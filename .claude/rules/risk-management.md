# リスク管理

## 最重要原則: 意味のある利幅を取れ

**2026-03-26実績: 勝率65%でNet -583円。勝ちが小さすぎて負けをカバーできない。**

- **ATR50%未満で切るな。** それはまだ動く余地がある。切るのはATR対比で意味のある利幅に達してから
- **「この利確額はスプレッド+手間に見合うか？」を自問しろ。** 見合わないなら切るな
- **切らない忍耐が利益を生む。** エントリーは悪くない(勝率65%)。利確が早すぎるのが全ての元凶

## 確度ベースのサイジング

**確度が高い→大きく張る。低い→小さく張る。**

| 確度 | 条件 | サイズ | 損切り幅 | 例 |
|------|------|--------|---------|-----|
| **S（鉄板）** | H1+H4+マクロ全一致、Div確認、ユーザー読み一致 | 5000-8000u | テーゼ崩壊まで | H4 ADX>30 + H1同方向 + M5押し目 |
| **A（高確度）** | H1方向一致 + M5タイミング確認 | 3000-5000u | H1構造変化まで | H1 bullish + M5 StochRSI=0.0 |
| **B（普通）** | 1TFのシグナルのみ | 1000-2000u | 状況判断 | M5 Divのみ、H1不明確 |
| **C（試し打ち）** | 根拠薄い | 500-1000u | 状況判断 | レンジ内逆張り |

**確度B以下を10回やるより確度Sを1回やれ。回数で稼ぐな。**

## add-on — 臨機応変に。ただし1つだけ絶対ルール

**ピラミッディングもナンピンもOK。状況次第。**

ただし: **「前回と違う根拠を言えるか？」** 言えなければ追加するな。
- 同じStochRSI=1.0が3回目 → 新情報ではない。それは「天井」ではなく「強いトレンド」
- 別TFで新しいシグナル、ニュース、サポートタッチ等 → 新根拠。追加OK

## 利確 — ATR対比で意味のある利幅を取れ

**「StochRSI=0.9だから半利確」→ NG。それはATRの何%か？テーゼの射程に対して何%か？**

判断基準（状況で使い分けろ）:
- **ATR対比**: 利幅がATRの50%未満なら切るな。まだ動く余地がある
- **テーゼ射程**: TP目標に対する進捗。まだ20%しか来てないのに切るのは早い
- **モメンタム変化**: MACDヒスト縮小+EMAスロープ鈍化+複数TFで同時に = 本当の利確サイン
- **含み益の大きさそのもの**: 300円以上あるなら確定する価値がある。40円を確定する意味はない

**利確しなさすぎも問題**: +2,833円→クローズアウトの教訓。含み益は幻。意味のある額になったら確定しろ。

## 損切り — テーゼベースで判断（金額で切るな）

**「-500円だから切る」→ NG。「テーゼが崩れたから切る」→ OK。**

- **損切り後に戻るパターンが多い**: テーゼが合ってるのにサイズの含み損に怯えて切る→戻る。これが一番の損失源
- **判断フロー**:
  1. H1構造が変わったか？ → NO → 切るな
  2. H1構造が変わった → 即全撤退
  3. 時間経過で戻らない → 状況判断で段階的撤退
- **SLなし裁量管理**: 固定SLはノイズで狩られる
- **スパイクが来ても慌てるな**: 天井/底で慌てて成行損切りが最悪。戻りを待て

## マージン管理
- **90%まで使ってよい**: marginUsed / NAV < 0.9 なら新規OK
- **新規禁止ライン**: marginUsed / NAV ≥ 0.9 → 新規禁止
- **強制半利確ライン**: marginUsed / NAV ≥ 0.95 → 最も含み損の大きいポジを即半利確
- **クローズアウト後30分待て**: 同じテーゼで即再エントリー禁止

## 失敗パターン（繰り返すな）

| パターン | 対策 |
|----------|------|
| 小さすぎる利確(+40円×13件) | ATR50%未満で切るな。スプレッド+手間に見合うか自問 |
| 利確遅延(+2,833→クローズアウト) | 意味のある含み益になったら確定。利確しすぎもしなさすぎもNG |
| 焦り損切り→戻る(GBP -237, EUR -246) | テーゼが生きてるなら切るな。金額ではなくテーゼで判断 |
| 同じ根拠でadd-on×7回 | 新しい根拠がなければ追加するな |
| 追っかけ(20本連続陽線で飛びつき) | 過熱検知=逆張りチャンス |
| スパイク慌て損切り(-3,832円) | 戻りを待て。天井/底で成行しない |
| H1思考停止(+353→+86) | MTFでモメンタム変化を検知して利確→回転 |
