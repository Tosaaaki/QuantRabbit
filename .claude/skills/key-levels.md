# キーレベルスキル

## 使い方

- 「UJレベル」→ USD/JPYのキーレベル一覧
- 「EUのフィボ」→ EUR/USDのフィボナッチ水準
- 「全ペアレベル」→ 全ペアの主要レベル

## 実行手順

### Step 1: データ取得

OANDA APIからキャンドルデータ:
- H4: 直近50本 (スイングハイ/ロー特定用)
- H1: 直近100本 (VWAP, S&R用)
- D1: 前日データ (ピボット計算用)

### Step 2: 計算

#### フィボナッチ (直近H4スイング)
直近の明確なスイングハイ/ローを特定し:
- 0% (スイングロー)
- 23.6%
- 38.2%
- 50.0%
- 61.8% ← 最重要
- 78.6%
- 100% (スイングハイ)

#### デイリーピボット
```
Pivot = (前日H + 前日L + 前日C) / 3
R1 = 2×Pivot - L
S1 = 2×Pivot - H
R2 = Pivot + (H - L)
S2 = Pivot - (H - L)
```

#### VWAP
H1の出来高加重平均価格（OANDA tick volumeベース）

#### S&R (水平線)
H1で複数回反発/反落した価格帯
