# 危険度スコアスキル

## 実行手順

### Step 1: データ取得

1. OANDA API: openTrades + 現在価格
2. H1/M5 ATR計算用キャンドル
3. shared_state.json: レジーム

### Step 2: 各ポジションのリスクスコア計算

| リスク要素 | 計算 | 配点 |
|-----------|------|------|
| SL/ATR比 | SL距離 ÷ H1 ATR。<1.0なら危険 | 0-25 |
| MTF逆行 | H1方向がポジと逆なら加点 | 0-25 |
| 含み損速度 | 直近5分の逆行pip/分 | 0-15 |
| 相関集中 | 同方向ポジの相関度合い | 0-15 |
| イベント近接 | 高インパクト指標が1時間以内 | 0-10 |
| マージン圧迫 | MarginUsed/NAV比率 | 0-10 |

**合計0-100点。高いほど危険。**

### Step 3: 出力

```
## ⚡ 危険度スコア

| ポジション | スコア | 状態 | 主要リスク |
|-----------|--------|------|-----------|
| AUD_USD S 1000u | 62/100 | 🟡警戒 | H1逆行+SL/ATR=0.8 |

### 詳細: AUD_USD SHORT
| 要素 | 値 | スコア |
|------|-----|--------|
| SL/ATR | 6.6pip / 8.0pip = 0.83 | 18/25 🔴 |
| MTF逆行 | H1=trending(up) vs SHORT | 20/25 🔴 |
| 含み損速度 | -5.4pip/5min | 8/15 🟡 |
| 相関集中 | 単独ポジ | 0/15 ✅ |
| イベント | BOE 1.5h後(間接) | 3/10 🟢 |
| マージン | 15.6% | 3/10 🟢 |

### 判定基準
| スコア | 状態 | アクション |
|--------|------|-----------|
| 0-30 | 🟢安全 | 通常管理 |
| 31-50 | 🔵注意 | SL確認 |
| 51-70 | 🟡警戒 | ロット縮小検討 |
| 71-85 | 🟠危険 | 即SL縮小 or 部分決済 |
| 86-100 | 🔴緊急 | 即全決済推奨 |
```
