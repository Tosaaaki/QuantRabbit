# イグジットアドバイザースキル

## 実行手順

### Step 1: データ取得

1. OANDA API: openTrades + 現在価格
2. shared_state.json: レジーム・テクニカル
3. キーレベル計算（TP/SL近接の水平線）

### Step 2: 各ポジション判定

判定軸:
- **テクニカル**: トレンド方向一致？ RSI極端値？ M5反転シグナル？
- **R:R進行**: Entry→現在の含み益 vs 残りTP距離。もう利益確保すべき？
- **時間**: 長時間ホールドで疲弊してないか（スキャルプ意図なのにスイング化）
- **リスク**: SL距離 vs 現在ボラ(ATR)。SLがATR以内なら危険
- **イベント**: 直近に高インパクト指標がある？

### Step 3: 出力

```
## 🚪 イグジット判定

### AUD/USD SHORT 1000u @0.70484
| 判定軸 | 状況 | 判定 |
|--------|------|------|
| テクニカル | H1 trending(上昇方向), M5 RSI=59.8 | ⚠️逆方向 |
| UPL | -27pip (含み損) | ❌不利 |
| SL距離 | 6.6pip (ATR=8pip) | ⚠️ATR以内 |
| TP距離 | 3.1pip | 🔶近い |
| 保有時間 | 5分 | ✅適正 |
| イベント | BOE 12:00 (GBP影響、AUD間接) | 🔶注意 |

### 💡 推奨: ⚠️ タイトSL監視
- 理由: H1がUP方向でSHORTは逆行中。ただしTP近接。
- 選択肢:
  1. **即クローズ** (-27pip) — H1逆方向を重視
  2. **TP到達を待つ** — あと3pipだが到達確率は低下
  3. **SLを1pip縮める** — リスク限定してTP待ち
```
