# ドローダウン警報スキル

## 実行手順

### Step 1: データ取得

1. OANDA API: 口座サマリー + openTrades
2. 各トレードのSL価格

### Step 2: 計算

#### 現在のDD
```
DD = (直近ピーク NAV - 現在NAV) / 直近ピーク NAV × 100
```

#### ワーストケース (全SLヒット)
各ポジションが全てSLに到達した場合の合計損失:
```
各ポジのSL損失 = |entry - SL| × units × pip単位 × JPY換算
合計ワーストケース = Σ(各SL損失)
口座比率 = 合計損失 / NAV × 100
```
