# 変更時の必須プロトコル

コード・設定・アーキテクチャを変更したら、以下を全て実行:

1. **CLAUDE.md更新**: アーキテクチャ変更時はルートのCLAUDE.mdを更新
2. **メモリ更新**: 該当するメモリファイルを更新
3. **変更ログ追記**: `docs/CHANGELOG.md` に追記
4. **mainにマージ**: ワークツリー編集時は必ずmainにマージ
5. **即デプロイ**: 変更したら即反映。聞くな
6. **日英同時編集**: プロンプトファイル（.claude/rules/、CLAUDE.md、scheduled-tasks/*/SKILL.md）を編集する際は、英語版（運用）と日本語版（参照用）を必ず同時に更新。内容は常に同期させる
