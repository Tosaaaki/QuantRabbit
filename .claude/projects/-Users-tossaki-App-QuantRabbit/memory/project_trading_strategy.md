## Claude裁量スキャルプ手法

前回成功セッション（2026-03-17 08:42-09:55, +978 JPY, WR80%）の手法を元に策定。

### 成功パターン（+978 JPY セッション）
- H1のV字回復パターンを認識 → ロング方向固定
- 73分で5トレード（14分に1回）= エントリー厳選
- 方向転換は1回のみ（ショート→ロング）
- 含み益は手動管理、自動TP/SLに頼らない

### 失敗パターン（-481 JPY セッション 10:17-12:04）
- RSI/EMAだけ見て機械的エントリー
- 100分で10回（10分に1回）= オーバートレード
- 方向ブレブレ（L→S→L→L→L→S→L→S→L→L）
- 自動TP/SLに放置

### 手法ルール
1. **チャートを読んでからエントリー**（H1→M5→M1の順で人間の目で判断）
2. **方向はH1で固定。ブレない**
3. **M5キャンドル完了後にエントリー判断**
4. **ノーマンズランド（EMA付近）では入らない**
5. **打診買い(1500u) → 確認 → 本玉(2500u)**
6. **TP: ATR×50%, SL: ATR×80%**
7. **含み益は手動判断で利確。自動TP任せにしない**

**Why:** 裁量（チャート読み）がボットや機械的ルールより圧倒的に勝率が高いことが2セッションで実証済み。
**How to apply:** セッション開始時にこのメモを参照し、必ずH1→M5→M1の順でチャートを読んでからトレード開始。

### 成功パターン2（2026-03-19 22時台, +482.8 JPY, W8/L1）
- JPY全面高の急騰相場でJPYクロスSHORT順張り
- swing-traderがAUD_USD SHORTで大きく取り(+438円)、scalp-fastは小さく刈り取り
- monitorの機械的partial利確（5段階）が感情排除して利益確定
- エージェント間で干渉なし（scalp-fastが「swing-trader's trade」とスキップ）
- **課題**: +482円の後に止められず、23時以降-234円吐き出し → 日次-1,215円

### 教訓（3/19追加）
- partial利確は機械(monitor)にやらせる。人間/Claude判断より優秀
- 他エージェントのポジに干渉しない → 正しい
- 利益が乗った後こそエントリー質を維持。焦り・退屈でのエントリーが利益を吐き出す原因（止めるのではなく質を保つ）
- 追撃はサイズ半分（EUR_USD 2本目750uでSL損失限定できた）
