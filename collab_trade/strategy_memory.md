## 🚨 致命的失敗: マージンクローズアウト (2026-03-26 03:30Z)

### 何が起きたか
- GBP_JPY SHORT 9本(9,500u) + AUD_JPY SHORT 12本(14,500u) + EUR_JPY SHORT 2本(2,000u) = 計23本
- AUD_JPYは一時+2,833円の含み益があった
- マージン残り2,935円の状態でEUR_JPY SHORT 2,000u @184.357を追加 → 逆行 → 全ポジション強制決済
- 結果: 直近30トレード合計 **-1,160円**

### 根本原因
1. **EUR_JPY 2000uが致命傷**: マージン極限で新規ペア追加 → 逆行でクローズアウト
2. **ナンピン自体は悪くなかった**: AUD_JPYのadd-onは効いて+2,833円まで行った。問題はマージン管理
3. **含み益を利確しなかった**: AUD_JPY +2,833円の時に半利確していれば、マージン余力ができてクローズアウトしなかった

### 教訓（v7ルール化済み）
- **marginAvailable < marginUsed × 0.3 → 新規エントリー禁止**
- **含み益+5pip以上のポジがあるのにマージンタイト → まず利確してマージン確保**
- **1ペアのadd-onは最大5本まで**（9本、12本は異常）
- **マージンクローズアウト後、同じテーゼで即再エントリーするな**（30分待て）

## メンタル・行動の教訓

- **慌てて損切りが最大の敵**: プロは慌てない。スパイクは5カテゴリで判断
- **損切り後に戻るパターンが多い**: テーゼが合ってるのにサイズ過大→含み損の金額に怯えて切る→戻る。**サイズを小さくすれば耐えられる。テーゼで判断しろ、金額で判断するな** (2026-03-26: GBP_JPY -237円→戻り、EUR_JPY -246円)
- **確度でサイズを決めろ**: S(鉄板)=5000-8000u、A(高確度)=3000-5000u、B(普通)=1000-2000u、C(試し)=500-1000u。確度B以下を10回やるより確度Sを1回やれ
- **小さい利確が最大の無駄**: 2026-03-26実績 — 勝率65%でNet-583円。勝ち平均+84円(13件が+50円以下)、負け平均-193円。ATR50%未満で切るな。「スプレッド+手間に見合うか？」を自問
- **エントリーは問題ではない。利確が問題**: 勝率は十分。利幅が小さすぎる。「切らない忍耐」が利益を生む
- **NY sessionのchopに振られるな**: 5-8pipの往来はNYでは正常。H1構造が崩れてなければ耐えろ(2026-03-23: UPL-288→+365の実績)
- **利確は8割で御の字**: +244→+54の利確遅延を繰り返すな
- **1ペアに固執するな**: 7ペアある。2連敗したら別ペアを探せ
- **マージン90%まで攻めろ**: チャンスがあれば積極的に。ただし10%は最低残す（クローズアウト防止）
- **記録は注文と同時**: 後回しにすると忘れる
- **ヘッドライン相場では追加LONG禁止**: 停戦→ultimatum→報復、30分で局面が変わる。trail保全が最優先
- **原油$10以上の急変動 = 全ペア影響**: WTI $89→$98の反発でセンチメント急変(2026-03-23)

## 両建て戦術（ヘッジ回転）

### マージンの仕組み（OANDA v20ヘッジ口座）
- **大きい方の片側にフルマージン。反対側は追加マージンゼロ**
- LONG 1000u持ち → SHORT 1000uまでマージン追加なし
- LONG 1000u + SHORT 500u → マージンは1000u分（大きい方）のまま
- つまり**既存ポジの逆側はタダで建てられる**

### MTF両建て — テーゼの階層分離
- **H1テーゼ: LONG** — 大局の方向。これは維持
- **M5/M15判断: SHORT** — 短期の過熱・プルバックで逆を取る
- テーゼは矛盾しない。タイムフレームが違うだけ
- **例**: EUR_USD H1 LONG(USD弱テーマ) + M5 StochRSI=1.0/RSI>70 → M5 SHORTで15-20pip回転 → プルバック終了でSHORT利確 → LONGだけ残る

### 使いどころ
| 場面 | 操作 | マージン |
|------|------|---------|
| M5過熱(StochRSI=1.0, RSI>70) | 同量以下ショート建て→下げで利確 | 追加なし |
| フリップ判断 | 先にショート建て→確認後にロング決済 | スリッページ回避 |
| ニュース前 | 同量ヘッジ→方向見えたら片方決済 | 追加なし |
| H1転換疑い | ショート先行→H1確定で判断 | 追加なし |

### 注意点
- **ファイナンスコスト**: 両側にスワップかかる（保有時間短ければ無視できる）
- **マージン増加ケース**: ショート側がロング側を超えたら、ショート側にフルマージン
- **AUD等ノーポジペア**: 新規は片側でもフルマージン必要。既存ペアのヘッジ利益で資金を作る

### 回転の典型パターン
1. EUR_USD LONG 1000u保有中、M5 StochRSI=1.0/CCI>200
2. SHORT 500u建て（マージン追加0）
3. M5プルバックで-10pip → SHORT +7pip利確（≒+1,100円）
4. LONG はH1テーゼ通りに保持
5. 利確分のマージンでAUD等に新規エントリー可能

- **根本原因**: H1テーゼ（Hidden Bullish MACD div + VWAP-10pip）がまだ生きていたのにM5だけで判断した
- **教訓**: **段階的撤退ルール** — M5割れ→5分ホールド(フェイクブレイク判定)→10分戻らず半分切り→20分+全撤退。-30pip/-500円超は即全撤退。H1テーゼは「すぐ切らない」理由にはなるが「ずっと持つ」理由にはならない
