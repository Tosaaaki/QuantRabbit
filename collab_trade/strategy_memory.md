## 両建て戦術（ヘッジ回転）

### マージンの仕組み（OANDA v20ヘッジ口座）
- **大きい方の片側にフルマージン。反対側は追加マージンゼロ**
- LONG 1000u持ち → SHORT 1000uまでマージン追加なし
- LONG 1000u + SHORT 500u → マージンは1000u分（大きい方）のまま
- つまり**既存ポジの逆側はタダで建てられる**

### MTF両建て — テーゼの階層分離
- **H1テーゼ: LONG** — 大局の方向。これは維持
- **M5/M15判断: SHORT** — 短期の過熱・プルバックで逆を取る
- テーゼは矛盾しない。タイムフレームが違うだけ
- **例**: EUR_USD H1 LONG(USD弱テーマ) + M5 StochRSI=1.0/RSI>70 → M5 SHORTで15-20pip回転 → プルバック終了でSHORT利確 → LONGだけ残る

### 使いどころ
| 場面 | 操作 | マージン |
|------|------|---------|
| M5過熱(StochRSI=1.0, RSI>70) | 同量以下ショート建て→下げで利確 | 追加なし |
| フリップ判断 | 先にショート建て→確認後にロング決済 | スリッページ回避 |
| ニュース前 | 同量ヘッジ→方向見えたら片方決済 | 追加なし |
| H1転換疑い | ショート先行→H1確定で判断 | 追加なし |

### 注意点
- **ファイナンスコスト**: 両側にスワップかかる（保有時間短ければ無視できる）
- **マージン増加ケース**: ショート側がロング側を超えたら、ショート側にフルマージン
- **AUD等ノーポジペア**: 新規は片側でもフルマージン必要。既存ペアのヘッジ利益で資金を作る

### 回転の典型パターン
1. EUR_USD LONG 1000u保有中、M5 StochRSI=1.0/CCI>200
2. SHORT 500u建て（マージン追加0）
3. M5プルバックで-10pip → SHORT +7pip利確（≒+1,100円）
4. LONG はH1テーゼ通りに保持
5. 利確分のマージンでAUD等に新規エントリー可能
