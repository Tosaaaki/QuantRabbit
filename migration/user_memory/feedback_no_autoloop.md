バックグラウンドの自動ループに任せない。自分で市況を見て、自分で判断してエントリーする。

**Why:** ユーザーから「ボットに任せてるのと同じ。怠慢」と指摘。Claudeの裁量判断力が価値なのに、ループに逃げたら意味がない。実際、ループがSHORT条件を探し続けてLONGの機会を逃した。

**How to apply:**
1. 数十秒ごとに自分でprice/RSI/momentum/EMAを取得して判断する
2. バックグラウンドコマンドで「自動エントリー」するループは禁止
3. 監視ループ（ポジション持ってる時のUPL追跡）はOKだが、エントリー判断は必ず手動
4. 毎回「なぜ入る/入らない」を言語化する
5. 状況が変わったら即座に方針を変える（ループでは方針変更が遅れる）
