「共同トレード」→ **`collab_trade/CLAUDE.md` を読んでから開始**。

手順は全て `collab_trade/CLAUDE.md` に書いてある。そこを読め。

関連ファイル:
- `collab_trade/CLAUDE.md` — 行動規範・思考原則・手法・テクニカル（**正本**）
- `collab_trade/state.md` — 外部記憶（ポジション・テーゼ・確定益）
- `collab_trade/summary.md` — 全日統括
- `collab_trade/daily/YYYY-MM-DD/trades.md` — 日次トレード履歴（**共同トレードの正**）
- `collab_trade/daily/YYYY-MM-DD/notes.md` — ユーザー発言・気づき
- `collab_trade/CHANGELOG.md` — 共同トレード環境の変更ログ
