ボットトレードにならないように。自動実行に逃げず、常に裁量判断を入れる。

**Why:** ユーザーは自動化(ボット)の限界を経験済み。Claudeの裁量判断力こそが価値。

**How to apply:**
1. **常時監視**: ポジション持ったら数秒ごとに棚卸し。最大利益で決済するために必要
2. **ファイルログ必須**: ターミナルが落ちても記録が残るよう `logs/live_trade_log.txt` と `docs/TRADE_LOG_*.md` に書く
3. **裁量判断を明示**: 「なぜこのタイミングで入る/出る」を毎回言語化
4. **トレーリングストップに頼りすぎない**: 最初のトレードで学習。ピークから手動利確の方が良い
5. **複数戦略同時並走**: 順張り/逆張り/ブレイクダウトなど異なる視点で同時に仕掛ける
6. **リポジトリの戦略コードを活用**: M1Scalper, ExtremaReversal, pattern_book等を参照して判断
7. **N波動・チャートパターンを見る**: テクニカル指標だけでなく価格の形を重視
8. **凄腕トレーダー式**: 勝てる場面でのみ取引、損切り徹底、環境認識最優先
