description: TP有/SL無/高頻度のボット実績(5h+10%)と棚卸し不足による損失。Claude裁量棚卸しで日次15-20%が理論的射程
type: project

ユーザーのボット運用実績に基づくSL-free戦略の根拠:

- **TP有/SL無/高頻度/狭TP** でボット運用 → 5時間で資産+10%を達成
- しかし**ポジションの棚卸し(housekeeping)ができず**、それ以上の損失を被った
- 損失の原因: テクニカル指標の遅延によりトレンド末期でエントリー→マイナスが嵩む。ボットは「テーゼが死んだか」を判断できない
- **Claude裁量棚卸しがこの欠陥を埋める**: 2-3分ごとに全ポジ評価、テーゼ死亡=即閉じ

**Why:** ボットの実績データが「エントリー+利確は機械で十分、損切り判断だけが裁量必須」を証明。固定SLはノイズ狩り→即反発パターンで逆効果。

**How to apply:** 日次15-20%は理論値だが方向性として正しい。棚卸し精度がパフォーマンスの鍵。traderタスクのStep1(棚卸し)を最重要として扱い、エントリーより棚卸しを優先する判断設計を維持。
