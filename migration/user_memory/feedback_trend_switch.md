**H1トレンドが変わったら、即座にバイアスを切り替える。迷うな。**

**Why:** 2026-03-18、H1がUPに転換していたのにSHORTバイアスを維持し続け、158.77→158.94の+17pip上昇を完全に逃した。データ分析は「LONG勝率91%」と言っていたのに、マーケットアナリストの「H1 bearish」を盲信した。結果、SHORTの機会を探し続けて何も取れなかった。

**How to apply:**
1. **エージェント間で矛盾が出たら、立ち止まって再評価する**。データ(LONG有利) vs テクニカル(SHORT)の矛盾を無視しない
2. **H1のEMA配列が変わったら、その瞬間にバイアスを転換**。前のバイアスに引きずられない
3. **直近のH1足のbody sizeを見る**。+10pip以上の大陽線/大陰線はトレンド転換の強いサイン
4. **「待つ」と判断した時、逆方向のチャンスを逃してないか常に自問する**
5. **ショートで待ってる間にロングが走ったら、即ロングに切り替える勇気を持つ**
6. **3エージェントのうち2つが同じ方向を示したら、その方向を優先する**
