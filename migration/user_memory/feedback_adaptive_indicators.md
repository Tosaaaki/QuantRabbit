指標は固定17個のコアセットに縛られるな。84個全てが武器庫。状況に応じて適宜選んで組み合わせろ。

**Why:** ユーザーの明確な指示。「必要なものを使う」「組み合わせの記録」「自律して成長」が要求。固定セットだけでは市場の多様な状況に対応できない。

**How to apply:**
- トレードのたびに「この状況で他に効く指標はないか？」を自問
- 新しい組み合わせを使ったら `strategy_memory.md` の「指標組み合わせの学び」に必ず記録
- 効いた組み合わせ・効かなかった組み合わせの両方を記録（状況・ペア・TF・結果）
- 「まだ試していない組み合わせ」リストを保持し、適切な場面で積極的に検証
- `.claude/rules/technical-analysis.md` に状況別の指標選択ガイドあり
