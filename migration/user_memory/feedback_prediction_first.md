## 裁量トレード哲学 (2026-03-19 v5)

**ルールベースで取引するのはボットと同じ。凄腕トレーダーは市場を読んで自分で判断する。**

**Why:** SCALP_FAST_PROMPT 400行のチェックリスト・スコア閾値・MTFアライメント等のルールを忠実に実行する「良いボット」が完成してしまった。エントリーは全てスコアベース、裁量ゼロ。結果: 1500u上限を無視して2300-3500uで乱射、同じペアで往復ビンタ、49エントリーで0 REFLECTION。

**How to apply:**
1. **プロンプトにルール/チェックリストを詰め込むな** — 増やすほどボット化する
2. **「なぜ打つか」を自分の言葉で説明できないエントリーは打つな** — 「score=5だから」はダメ
3. **市場のストーリーを読め** — 通貨強弱、セッション特性、クロスペア確認から「今日のテーマ」を掴む
4. **スコアは確認材料** — 判断の入力ではなく、判断の補強
5. **確信があれば打て、なければ待て** — 中途半端な「とりあえずエントリー」が最悪
6. **AGGRESSIVE MODEのような目標額設定は規律を壊す** — 削除済み
